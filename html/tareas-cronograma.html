Tareas y Cronograma

Este apartado recoge la planificación del proyecto organizada por fases del ciclo de desarrollo de software. Cada fase incluye las tareas realizadas, las fechas de ejecución, la estimación de esfuerzo en horas y los entregables asociados.

Fase 1 Planificación y diseño 20 - 28 Nov 2025 40 h
Depende de: — Entrega: Modelo de datos, Casos de uso
  • Formar equipos y firmar contrato de aprendizaje
  • Analizar el reto y definir la temática
  • Crear diagrama de casos de uso
  • Diseñar el modelo de datos
  • Diseñar la base de datos relacional
  • Planificar la base de datos PostgreSQL
  • Configurar Odoo Projects
  • Definir paquetes de trabajo y asignar responsables
  • Registrar diariamente el tiempo en Odoo
  • Investigar alternativas de temática
  • Presentar y seleccionar la propuesta
Fase 2 Diseño detallado 1 - 12 Dic 2025 30 h
Depende de: Fase 1 Entrega: Diagrama de navegación, Diseño detallado
  • Diseñar el diagrama de navegación
  • Especificar el diseño detallado de cada vista
  • Diseñar vistas web para Cliente
  • Diseñar el esquema de colecciones MongoDB
  • Planificar la estructura del servidor PHP
🟡 Vacaciones de Navidad: 20 Dic 2025 — 7 Ene 2026
Fase 3 Construcción y prueba 8 Ene - 16 Feb 2026 96 h
Depende de: Fase 2 Entrega: Aplicación completa, informes, módulos
Sprint 1 Fundaciones y autenticación 8 - 16 Ene 40 h
  • Implementar la base de datos MySQL
  • Crear los enums y la clase CartItem
  • Implementar la interfaz DAO y DBConnection
  • Configurar hibernate.cfg.xml
  • Crear el script SQL con datos de prueba
  • Configurar el proyecto JavaFX
  • Implementar la ventana de LogIn
  • Comenzar el módulo de Odoo
  • Configurar la base de datos PostgreSQL
Sprint 2 Registro y navegación 16 - 22 Ene 24 h
  • Implementar el registro de usuarios
  • Implementar el menú principal
  • Implementar la ventana de tienda / catálogo
  • Implementar lógica de autenticación
  • Crear ventana de ayuda
  • Desarrollar vistas del módulo Odoo
Sprint 3 Carrito, reseñas y pedidos 22 - 29 Ene 20 h
  • Implementar el carrito de compras
  • Implementar ventana de reseñas
  • Implementar la ventana de modificación de perfil
  • Implementar pedidos
  • Añadir validaciones en todos los formularios
Sprint 4 Panel de administración 29 Ene - 3 Feb 20 h
  • Implementar el panel de administración
  • Implementar añadir videojuegos
  • Implementar modificar videojuegos
  • Implementar eliminar cuentas de administrador
  • Añadir ThreadConnection para conexiones simultáneas
Sprint 5 Integración y empaquetado 3 - 9 Feb 20 h
  • Integrar JasperReports en JavaFX
  • Empaquetar la aplicación JavaFX
  • Implementar la aplicación web Cliente
  • Implementar el servidor PHP con servicios REST
  • Completar el módulo Odoo con CRUD completo
Sprint 6 Migración y cierre 9 - 16 Feb 12 h
  • Migrar la aplicación JavaFX a MongoDB
  • Analizar la estructura actual y diseñar migración
  • Implementar prueba final y revisión
