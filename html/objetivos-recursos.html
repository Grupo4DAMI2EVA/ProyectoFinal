Objetivos y Recursos

Planificación estratégica y bases tecnológicas del proyecto Desktop.

Objetivo General del Proyecto

A la hora de gestionar una tienda de videojuegos con un catálogo amplio de productos y datos de usuarios, es de vital importancia una buena organización para evitar errores y confusiones. Por esa misma razón se plantea una aplicación de escritorio para gestionar el trabajo y facilitar las tareas lo máximo posible.

El sistema ofrece una experiencia dual: por una parte, la vista del usuario para consulta de catálogo (título, género, plataforma, precio, etc.), gestión de pedidos y reseñas. Por otro lado, un entorno administrativo potente que permite el control total (CRUD) del catálogo y la gestión de cuentas.

Objetivos Específicos y Tecnologías

Este proyecto se consolida como una solución robusta basada en el ecosistema Java:

  • Stack Tecnológico: Desarrollo con JavaFX para la interfaz, MySQL como motor de base de datos y Hibernate (ORM) para la persistencia.
  • Niveles de Acceso: Implementación de roles User y Admin con procesos de autenticación y registro (Sign Up) seguros.
  • Funcionalidades de Usuario: Filtrado avanzado mediante HQL, gestión de carrito de compras, sistema de pedidos y reseñas interactivas.
  • Herramientas Administrativas: Control total de inventario y usuarios, además de la generación de informes profesionales en PDF mediante JasperReports.
